# Team Research Archive

This folder contains research notebooks and supporting files contributed by
group members during the V2 recommender exploration. **These files are NOT
used at runtime.** They are archived here for reference and for the project
defence.

## Why these are not in production

The notebooks operate on `cleaned_movies_metadata.csv` — an English-only
subset of ~5,363 films — whereas the main app uses `data/processed/movies_clean.csv`
which contains 22,620 films. The similarity matrices produced by these
notebooks (110 MB each, originally `SBERT_similarity_matrix.pkl`,
`tag_similarity_svd.pkl`, `tag_similarity_onehot_cosine.pkl`) have
**incompatible dimensions** with our movie set and cannot be looked up by
the TMDB IDs we use elsewhere. Switching the entire app to the 5,363-row
dataset would have cascaded through `analysis.py`, the poster lookup, and
every dashboard chart — a poor trade for the marginal quality gain over the
TF-IDF approach now shipped in `recommender.py`.

The three `.pkl` files were therefore deleted (330 MB total). They can be
regenerated by re-running the notebooks below.

## Files

| File | Purpose |
|------|---------|
| `content_based_SBERT.ipynb` | Builds Sentence-BERT embeddings of movie overviews and computes a 5,363 × 5,363 cosine similarity matrix. |
| `genre_SVD.ipynb` | One-hot encodes genres, applies TruncatedSVD, computes a genre-similarity matrix. |
| `unique_genre_similarity.ipynb` | Simpler one-hot genre cosine similarity, with sampling. |
| `clean_ratings.py` | Maps MovieLens `movieId` → IMDB `imdbId` via the team's `links.csv`. Not used in the runtime pipeline. |
| `rating_table.csv` | 5,358-row enriched metadata table with VADER sentiment scores (`sentiment_compound`, `sentiment_score`, `sentiment_category`, `review_count`). |
| `links_team_version.csv` | Team's copy of the MovieLens ID-bridge file (45,844 rows). The production app uses `data/raw/links.csv`. |

## Production recommender

The shipping recommender lives in `recommender.py` (project root) and uses
TF-IDF + cosine similarity over the English-language subset of
`movies_clean.csv`. See `CLAUDE.md` and the commit message for details.
